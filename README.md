# 仮想通貨相場トラッカー

#### url : http://honghyunAhn.github.io/react-master

## 機能

- React Query を利用して Crypto Price API のデータを取得。

- Styled Components を利用して拡張可能なデザイン機能

- ダークモード機能

- Apex Charts を活用して仮想通貨の相場変化データを視覚化

## Using

- Styled Components
- Recoil
- React Query
- React Router DOM
- Apex Charts
- React Spring
- Typescript
- Firebase
